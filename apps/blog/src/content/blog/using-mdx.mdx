MDX is a special flavor of Markdown that supports embedded JavaScript & JSX syntax. This blog uses [next-mdx-remote](https://github.com/hashicorp/next-mdx-remote) to render MDX content with full support in the Next.js App Router.

## Why MDX?

MDX unlocks the ability to mix JavaScript and UI Components into your Markdown content for things like interactive charts or alerts.

## Example

Here is a simple interactive element written inline in MDX:

<a href="#" onClick={() => alert('clicked!')}>
  Embedded component in MDX
</a>
